# Nhiem vu tung thu muc

## frontend
Chua toan bo giao dien client-side bang HTML, CSS va Vanilla JavaScript.

## frontend/admin
Khu vuc giao dien quan tri. Tat ca trang admin dung chung layout va component.

## frontend/admin/components
Chua component tai su dung nhu sidebar, header va footer. Khong copy HTML/CSS/JS giua cac trang.

## frontend/admin/dashboard
Trang tong quan so lieu va trang thai cua he thong.

## frontend/admin/products
Trang quan ly san pham. Chua viet CRUD o giai doan nay.

## frontend/admin/categories
Trang quan ly danh muc san pham.

## frontend/admin/orders
Trang quan ly don hang.

## frontend/admin/users
Trang quan ly nguoi dung va tai khoan.

## frontend/admin/payments
Trang quan ly thanh toan.

## frontend/admin/vouchers
Trang quan ly ma giam gia.

## frontend/admin/emails
Trang quan ly email dang ky nhan tin.

## frontend/admin/settings
Trang cau hinh he thong.

## frontend/assets
Tai nguyen dung chung cho frontend: CSS, JS, hinh anh va icon.

## backend
Chua server Node.js Express theo MVC.

## backend/config
Cau hinh ung dung, database va bien moi truong.

## backend/controllers
Nhan request va dieu phoi service. Chua viet controller o giai doan nay.

## backend/models
Lop truy cap du lieu MySQL. Chua viet model o giai doan nay.

## backend/routes
Khai bao route Express. Chua viet API o giai doan nay.

## backend/middleware
Middleware dung chung nhu auth, upload, error handler sau nay.

## backend/services
Xu ly nghiep vu chinh, nam giua controller va model.

## backend/utils
Helper dung chung, format du lieu, validate, response helper.

## backend/uploads
Noi luu file upload trong qua trinh phat trien.

## backend/public
Static files duoc Express phuc vu neu can.

## database
Tai lieu database, migration va seed SQL sau khi duoc xac nhan.

## docs
Tai lieu kien truc, quy uoc code, cau truc thu muc va phan tich nghiep vu.
