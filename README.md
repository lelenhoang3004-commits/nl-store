# Fashion Store Website

Du an website ban hang thoi trang theo kien truc MVC, tach rieng frontend, backend, database va tai lieu.

Trang thai hien tai: chi tao cau truc thu muc va file nen tang. Chua viet CRUD, API hoac SQL.

## Cong nghe

- Frontend: HTML5, CSS3, Vanilla JavaScript ES6
- Backend: Node.js, Express.js
- Database: MySQL
- Architecture: MVC

## Huong phat trien

1. Hoan thien UI component cho Admin Panel.
2. Thiet ke database MySQL sau khi thong nhat nghiep vu.
3. Xay dung model, service, controller va route theo tung module.
4. Ket noi frontend voi backend bang API rieng sau khi duoc xac nhan.
